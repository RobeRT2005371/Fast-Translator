# FastTranslate

FastTranslate is a fast, responsive multilingual translation app with real-time speech synthesis and copy-to-clipboard features.  
It is designed for usability, accessibility, and speed, making translation seamless across devices.

---

##  Overview
FastTranslate enables users to:
- Translate text instantly between multiple languages
- Hear translations spoken aloud
- Copy translated text to the clipboard with one click
- Use a clean, responsive interface optimized for desktop and mobile

---

##  Features
-  Real-time multilingual translation
-  Speech synthesis for translated text
-  One-click copy-to-clipboard
-  Responsive design for mobile and desktop
-  Fast performance with lightweight architecture
